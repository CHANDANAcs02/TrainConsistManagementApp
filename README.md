# Train Consist Management App

A simple Java console application that demonstrates Train Consist Management using Java programming concepts.

## Features

- Train initialization
- Passenger bogie management
- Unique bogie IDs
- Ordered train consist
- Insertion order using LinkedHashSet
- Bogie-capacity mapping
- Sorting and searching algorithms
- Stream API operations
- Regular expression validation
- Exception handling
- Performance comparison
- Bubble sort and binary search

## Technologies

- Java
- IntelliJ IDEA
- Git & GitHub
